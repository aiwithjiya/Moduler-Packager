# 🚀 Multi-Utility Toolkit 

## 📌 Project Overview

**Multi-Utility Toolkit** is a Python-based utility application that provides multiple useful tools in a single menu-driven program.

This project is developed using Python built-in modules such as **datetime, time, math, random, uuid, and os**. It demonstrates the practical implementation of different Python functionalities including date and time operations, mathematical calculations, random data generation, unique identifier generation, file handling, and module exploration.

The main objective of this project is to understand and apply Python standard libraries to create a simple, interactive, and user-friendly utility toolkit.

---

## ✨ Project Features

### 🕒 Datetime & Time Operations
- Display current date and time
- Calculate difference between dates
- Format dates into custom formats
- Stopwatch functionality
- Countdown timer

### 🔢 Mathematical Operations
- Factorial calculation
- Compound interest calculation
- Trigonometric calculations
- Geometrical area calculation

### 🎲 Random Data Generation
- Random number generation
- Random selection from list
- Random password generation
- OTP generation

### 🆔 UUID Generation
- Generate unique identifiers using UUID module

### 📁 File Operations
- Create files
- Write data into files
- Read file content
- Append data into files

### 🔍 Module Exploration
- Explore Python built-in module attributes using `dir()`
